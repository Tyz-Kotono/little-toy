# XToolkit v1.2 Release

## 🎉 新版本发布

### ✨ 主要功能
- **多平台支持**：YouTube、Twitter 视频下载
- **GIF 转换**：Twitter 视频可转换为 GIF 动图
- **动态预览**：GIF 文件动态播放预览
- **Dock 布局**：预览窗口以 Dock 形式显示
- **进度显示**：实时下载进度条

### 🔧 技术特性
- 模块化设计：YouTube 和 Twitter 独立模块
- PyQt6 界面：现代化桌面应用
- 多线程下载：不阻塞界面
- 自动检测：YouTube 画质和字幕检测

### 📦 安装说明
1. 下载 `XToolkit.exe`
2. 直接运行即可（无需安装 Python 环境）
3. 首次使用建议安装 FFmpeg（用于 GIF 转换）

### 🚀 使用方法
1. 输入视频链接
2. 选择平台（默认 Twitter）
3. 配置下载选项
4. 选择保存位置
5. 开始下载

### 📝 更新日志

- v1.1：添加进度条、GIF 转换功能
